# API Overview

This section outlines publicly available or internal APIs and how to authenticate.

## Auth

- Most API endpoints require a bearer token (Supabase session or API key depending on the environment).

## Key Endpoints

- `GET /api/list-resumes` — list user resumes
- `POST /api/upload-resume` — upload resume file
- `POST /api/send-application` — send an application to a job

For programmatic integrations, contact the team to request API access and production keys.
