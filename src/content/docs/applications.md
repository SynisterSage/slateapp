# Applications & Tracking

Track all jobs you've applied to and follow up efficiently.

## Application Records

- Each application includes: job reference, resume used, status, notes, and last activity.
- Statuses: Applied, Interviewing, Rejected, Offer.

## Notes & Activity

- Add notes to an application to track outreach, interview preparations, and outcomes.
- Use the timeline view in the application detail to see activity and messages.

## Exporting

- You can export application history for reporting or personal records. Exports include timestamps and linked job sources.
